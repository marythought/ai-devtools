App Features

This TODO application helps you organize and manage your tasks efficiently.

  • Create, edit, order and delete TODO items
  • Organize tasks with categories
  • Set due dates and effort levels
  • Track completion status
  • Drag-and-drop reordering
  • Multi-language support (English & Spanish)
  • Read-only demo mode

Technology Stack

  • Process: Built in a few hours with assistance from Claude Code. You can view the source code on Github.
  • Backend: Django (Python)
  • Frontend: HTML, CSS, JavaScript
  • Database: SQLite
  • DevOps: CI/CD support via Github Actions and Python Anywhere
